(W|H)all of Lame
================

Unencrypted data snippet (passwords,sessions,..etc) gathering via open wireless networks

Inspired by Wall of Shame/Wall of Sheep @ defCON


### Install

dependencies: `python tcpdump tshark aircrack-ng`


### Usage

```
$ ./whol.sh --help
# e.g.
$ sudo ./whol.sh -i wlan0
```
